## Installation 
Installing Note Takker on your computer is as easy as cloning the project repository. 
When the project has been cloned cd into the backend and run ```npm install``` in your terminal to install the backend dependencies.
```
 git clone https://github.com/k-dodsonknapp/NoteTaker.git
 cd backend 
 npm install
 npm start
 
```
Once the project has been cloned cd into the frontend and run ```npm install``` in your terminal to install the frontend dependencies.
```
 cd frontend  
 npm install
 npm start
```
To use the application in a development environment use ```npm start``` in the backend and in the frontend to start the application.
